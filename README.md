## Features

- **Neural Amp Modeling** - AI-powered amp simulation supporting multiple model types (ConvNet, LSTM, WaveNet)
- **Integrated Pedal Suite**
  - **Tube Screamer** - Classic overdrive with drive, tone, and level controls
  - **Compressor** - Transparent dynamics processor with attack and sustain control
  - **Clean Boost** - Transparent boost stage with presence shaping
- **Tone Stack** - 3-band parametric EQ (Bass, Middle, Treble)
- **Additional Processors**
  - Noise Gate for controlling feedback
  - Doubler effect for depth and width
  - Independent input/output gain control
- **Preset Management** - Save and load custom amp configurations
- **Real-time Metering** - Input/output level visualization
- **Multi-page Interface** - Organized effect layout (Pre-Effects, Amp, Post-Effects)
